About

This is a simple app to demonstrate Flask, WTForms, and Flask-Bootstrap.

Flask is a micro web framework written in Python.

WTForms is a flexible forms validation and rendering library for Python.

Flask-Bootstrap is a package that makes it easy to use Bootstrap in Flask applications.

Combining these technologies allows for rapid development of web applications with a modern and user-friendly interface.
